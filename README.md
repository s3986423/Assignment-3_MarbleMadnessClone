# Assignment 3: Marble Madness 3D Clone

## Features

* **Tileable World Design** – Levels built with Godot’s GridMap system for modular and reusable environments.
* **Three Progressive Levels** – From introduction to challenge to final gauntlet, each stage increases in complexity and intensity.
* **Dynamic Hazards** – Spike rollers, swipers, swinging hammers, and more to test player reflexes and precision.
* **Checkpoints & Goals** – Integrated system for respawning, progress tracking, and level completion.
* **Custom Physics Integration** – Hybrid system combining Jolt’s collision handling with custom gameplay mechanics for tight controls.
* **Replayability** – Multiple paths, skips, and time-based ratings encourage experimentation and mastery.

---

## Controls

* **Arrow keys / WASD** – Move
* **Space** – Jump
* **Enter** – Respawn immediately
* **Mouse input** - Camera Controls
* **Esc** – Pause

---

## How to Play

1. Download the repository.
2. Import the project **assignment-3/project.godot** in **Godot 4.2+**.
3. Run project (F5) to start the game.

Or download release build (.exe file) to run the game directly on Windows

---

## Development Setup

* Engine: **Godot 4.4.1 Stable**
* Language: **GDScript**
* Project structure:

  * `/Assets` → meshes, sounds, background
  * `/Charaters` → player
  * `/Components` → environments, obstacles
  * `/Stages` → level design
  * `/System` → manage stages
  * `/UI` → start, pause, end game buttons and scene
